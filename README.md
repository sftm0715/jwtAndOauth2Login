# jwtAndOauth2Login

### ✅ 프로젝트 소개 
OAuth2 Login 및 스프링 시큐리티 + JWT를 이용한 자체 Login 을 구현한 프로젝트 입니다.

### ✨ 구현 기능
- 스프링 시큐리티 + JWT 를 이용한 자체 회원가입/로그인/로그아웃
- OAUTH2 회원가입/로그인/로그아웃 : google, kakao, naver 소셜 회원가입 및 로그인

### 🛠 기술 스택
 ```
1. Spring Boot
2. Spring Data JPA
3. Spring Security
4. JWT
5. OAuth2.0
5. H2
 ```
